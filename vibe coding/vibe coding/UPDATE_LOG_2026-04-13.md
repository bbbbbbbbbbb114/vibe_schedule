# 更新日志（2026-04-13）

## 今日概览

## 功能新增
- 日程类型支持：时间点与时间段双模式。
- 可视化日历支持：
  - 时间点与时间段差异化展示。
  - 事件颜色高亮。
  - 重复事件展开显示（daily/weekly）。
  - 当前时间红线（now indicator）。
- 日程配置新增：
  - 地点标签（选填）。
  - 提醒提前量（5m/10m/1d）。
  - 提醒时机（start/end/both）。
  - 重复规则（none/daily/weekly + weekly weekdays）。
- 强提醒能力升级：
  - 0级弹窗（高优先级遮罩）。
  - 必须确认后关闭。
  - 提示音改为单次播放。
